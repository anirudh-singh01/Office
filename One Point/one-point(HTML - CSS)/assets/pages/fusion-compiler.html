Fusion Compiler

Next-generation RTL-to-GDSII design platform for advanced node designs

Overview

Fusion Compiler is a comprehensive RTL-to-GDSII design platform that delivers superior quality of results (QoR) and faster time-to-results for advanced node designs.

Key Features

  • RTL synthesis and optimization
  • Place and route capabilities
  • Advanced node support
  • Machine learning optimization
  • Multi-corner multi-mode analysis
