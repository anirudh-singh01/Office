VC SpyGlass

Comprehensive RTL design analysis and verification

Overview

VC SpyGlass provides comprehensive RTL design analysis and verification capabilities, helping identify and fix design issues early in the design cycle.

Key Features

  • RTL linting and analysis
  • Design rule checking
  • Clock domain crossing analysis
  • Power analysis
  • Design quality metrics

Status

Available

This tool is currently available for use in the Synopsys.ai Copilot platform.
